# Project Description

This project is about learning how to build, train, analyze, and modify a deep network for a recognition task. We will be using the MNIST digit recognition data set, primarily because it is simple enough to build and train a network without a GPU, but also because it is challenging enough to provide a good example of what deep networks can do.

-------------------------------------------------------------------------------------
# File Structure

- Task1AtoD.py - creates a cnn network and recognizes the digits and saves the model.
- Task1E.py - reads and runs the network on a test data set.
- Task1F.py - reads and runs the network to detect digits handdrawn/handwritten.
- Task2.py - examine the network created in task 1.
- Task3.py - recognize 3 different greek letters - alpha, beta, gamma. This script also detects the handwritten greek letter.
             This scripts also creates a modified Cnn netork with new dimensions (Extension 1).
- Task4.py - used MINSIT Fashion Dataset, using tensor flow, gives best config values.
- Task3_Extenstion.py - This script is for extension 2, it recognize 6 different greek letters - alpha, beta, gamma, delta, theta and eta. 
			This script also detects the handwritten greek letter.
- Pre-trained_EXT_4.py - Uses pre-trained model and evaluates and visualizes the first 3 convlution layers.
- GaborFilterNetwork.py - Replace the first layer of the MNIST network with a gabor filter.
- LiveDigitDetection.py - live video digit recognition application using the trained network.
